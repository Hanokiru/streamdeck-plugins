## Summary
- Yandex Alice Home **v0.3.0.0**: scenarios, device toggle, dials, color, modes, rooms/groups, sensors, quick presets
- Mana-style icons, bilingual docs (RU/EN), MIT © Hanokiru
- Package: `com.hanokiru.yandex-alice.streamDeckPlugin`

## Install
1. Download `com.hanokiru.yandex-alice.streamDeckPlugin` from this release
2. Double-click to install in Stream Deck
3. Create Yandex OAuth app with scopes `iot:view` + `iot:control`
4. Paste `access_token` into any Alice action settings
5. Press ↻ to load devices / scenarios

Full guides: [README (RU)](https://github.com/Hanokiru/streamdeck-plugins/blob/main/plugins/yandex-alice/README.md) · [README (EN)](https://github.com/Hanokiru/streamdeck-plugins/blob/main/plugins/yandex-alice/README.en.md)
